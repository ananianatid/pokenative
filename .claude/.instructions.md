# Pokenative AI Customizations

This file registers custom skills for the pokenative project to help AI agents scaffold components, manage API endpoints, and maintain styling consistency.

## Available Skills

### `/create-pokemon-component`
Scaffold domain-specific Pokemon UI components (stat rows, type badges, move lists) with correct styling patterns and TypeScript conventions.

**Trigger phrases:**
- "Create a Pokemon component"
- "Add a stat display"
- "Build a type badge"
- "Make a move list component"

### `/add-api-endpoint`
Add new PokéAPI endpoints with proper TypeScript types to the `API` generic type system in `hooks/useFetchQuery.tsx`.

**Trigger phrases:**
- "Add a new API endpoint"
- "Register a PokéAPI endpoint"
- "Add move/ability/item endpoint"
- "Extend the API type system"

### `/style-component`
Generate properly-typed StyleSheet constants following the `satisfies ViewStyle` pattern and project styling conventions.

**Trigger phrases:**
- "Style this component"
- "Create component styles"
- "Apply theme colors"
- "Generate a stylesheet"
