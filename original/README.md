# Pokenative - Version originale

Application Pokédex Pokémon construite comme projet d'apprentissage React Native.
Consultez le README racine pour l'organisation complète du projet.

## Stack technique

- React Native / Expo SDK 54
- Expo Router 6
- TanStack React Query
- TypeScript

## Lancer

```bash
cd original
npm install
npx expo start
```
